Perfil de egreso

La formacion que pfrece la carrera de Técnico en programación permite al egresado, a tráves de la articulación de saberes de diversos campos, realizar actividades dirigidas a la:

  • Instalación y desarrollo de software de aplicación utilizando programación estructurada y orientada a objetos en ambientes web y móviles, con almacenamiento persistente de datos,
  • Así como la configuración y administración de plataforma e-learning y
  • Comercio electrónico.
Durante el proceso de formación de los cinco módulos, el estudiante desarrolla o reforzará:
Las siguientes competencias profecionales:
  • Desarrolla e instala software de aplicación utilizando programación estructurada, con almacenamiento persistente en los datos
  • Desarrolla software de aplicacion utilizando programación orientada a objetos, con almacenamiento persistente en los datos
  • Desarrolla plicaciones web y móviles
  • Administra sistemas operativos, aplicaciones y servicios
  • Desarrolla, administra y configura soluciones de e-learning y comecrio electrónico
